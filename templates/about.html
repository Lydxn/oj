Frequently Asked Questions

Who created this website?

It was created by Lyndon Ho for a Grade 12 final project.

What is this website?

This is a programming website designed to help both novice and experienced programmers improve their coding skills! The idea for this site was inspired by CodingBat and DMOJ. Right now it is under beta development, and will see more cool awesome features in the near future!

How do I use this website?

Click on the Problemset at the top of this page, and select one of the problems from the list that are available. Each problem will contain a problem statement that you can solve and earn internet points for!

What do the statuses mean?

The status for a given submission or test case can mean one of the following things:

  • AC - Accepted (Solution is correct!)
  • WA - Wrong Answer (Solution is not correct)
  • MLE - Memory Limit Exceeded (Solution used up too much memory)
  • TLE - Time Limit Exceeded (Solution took too long to run)
  • NZE - Non Zero Exit (Solution terminated with a non-zero exit code)
  • RE - Runtime Error (Solution terminated with a signal)
  • CE - Compile Error (Solution did not compile)
  • IE - Internal Error (An internal error occurred while judging)
